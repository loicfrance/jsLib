getVertices(verticesArray, vOffset, indicesArray, iOffset)

draw the shape on the canvas using its webgl context. To fill the shape, the draw mode must de TRIANGLES. you can acquire how many points will be added to the array or how many triangles will be drawned by using the attributes utils.geometry2d.Point#glPointsNumber and utils.geometry2d.Point#glTriangles
