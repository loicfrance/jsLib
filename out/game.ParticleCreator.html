created  number

number of particles created. equal to -1 if the creator is paused. if the max attribute is equal to 0, this value is clamped between 0 and 1 to keep a high precision for creating particles continuously. When the creator is restarted, this value is set to 0.

max  number

maximum number of particles this creator can create in its lifetime. When this number is reached, the creator stops creating particles and when no particles are living anymore, the creator kills itself. If this value is equal to 0, the maximum is disabled : the creator will continuously create particles until the game is over or the value changed.

renderLayer  unknown

the render layer of this object. As it is generally common to all objects of the same class, this attribute is generally defined in the prototype of the class, but you can do it anywhere you want for your objects. The default value is game.RenderLayer.OBJ1; This attributes is used to define an order for the objects to be drawn on the screen, and must be part of the game.RenderLayer enumeration, or an other number if you handle it in your own game map.
