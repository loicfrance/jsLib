the render layer of this object. As it is generally common to all objects of the same class, this attribute is generally defined in the prototype of the class, but you can do it anywhere you want for your objects. The default value is game.RenderLayer.OBJ1; This attributes is used to define an order for the objects to be drawn on the screen, and must be part of the game.RenderLayer enumeration, or an other number if you handle it in your own game map.
