game.objectProperties.health

Source: game2/object_mixins.js:17

provides health methods and properties to create objects such as enemies, players, destructible objects, ... adds the health attribute to the object, with a default value of 0 added as a prototype property, and a maxHealth optional attribute (not created by default, but used in method if it exists). also adds getters and setters for those two attributes, and two other methods : heal and receiveDamages
