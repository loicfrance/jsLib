The class used by the game manager for the rendering.

Properties

context  (CanvasRenderingContext2D or WebGLRenderingContext)

abstract

render

called by the game manager after a frame to draw objects on the canvas. The callback function is called at the beginning and at the end of the function, and at the beginning and the end of the drawing of every layer, with the first parameter equal to game.RenderEvent.RENDER_BEGIN, game.RenderEvent.RENDER_END, game.RenderEvent.RENDER_LAYER_BEGIN or game.RenderEvent.RENDER_LAYER_END <!-- and the second one equal to the rendering context.

setCanvasSize(width, height, marginX, marginY)

manually changes the size of the canvas, and modifies the scale for the visible rectangle to fit the canvas without the visible rectangle to change. you can change the visible rect and the game.Viewer#updateScale method to change the scale. The callback function is then with the first parameter equal to game.RenderEvent.CANVAS_RESIZE and the second one equal to the rendering context.

useAutoResize(use[, borderMargin])

allows the canvas to resize automatically when the window size changes. The callback function is called after the resize with the first parameter equal to game.RenderEvent.CANVAS_RESIZE and the second one equal to the rendering context.
