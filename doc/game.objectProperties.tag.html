game.objectProperties.tag

Source: game2/game_2.js:1109

provides tag methods and property. when needed, adds a tags attribute that will contain all future tags associated with the object. adds the following methods :

  • addTag that adds the specified tag to the current ones. creates the tags attribute if needed,
  • hasTag that checks if the object has the specified tag,
  • getTags that returns the tags attribute,
  • isTagged that returns true if the object has at least one tag,
  • clearTags that clears all tags of the object by re-creating the tags attribute.

You can also use some function such as game.objectProperties.tag_canHaveTag and game.objectProperties.tag_hasTag, that you can use as a filter for objects, and game.objectProperties.tag_getAllObjectsWithTag.
