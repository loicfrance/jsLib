getIntersection(rects) → utils.geometry2d.Rect

returns the intersection of the given rectangles, i.e. the rectangle formed by the maximum left and top, and the minimum right and bottom of all rects. if the max left(resp. top) happen to be higher than the minimum right(resp. bottom), or if the given array is null, this function returns null.
